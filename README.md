# vharyani07.github.io

Personal portfolio for **Vivak Haryani**.

## Deploy (GitHub Pages)

1. Put these files in your `vharyani07.github.io` repository root:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `assets/Vivak_Haryani_Resume.pdf`

2. In GitHub:
   - Repo **Settings → Pages**
   - **Build and deployment:** Deploy from a branch
   - Branch: `main` • Folder: `/(root)`

3. Your site will be live at: `https://vharyani07.github.io`

## Customize

- Update copy in `index.html` (Hero/About/Experience)
- Add screenshots and link them in the Projects cards
